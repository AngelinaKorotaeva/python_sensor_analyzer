# Sensor Analyzer

Python project for analyzing sensor temperature data from TXT and CSV files.

## Features

* Read sensor data from TXT files
* Read sensor data from CSV files
* Calculate maximum temperature
* Calculate minimum temperature
* Calculate average temperature
* Generate sensor reports
* Save reports to files

## Technologies

* Python
* File handling
* CSV processing
* Functions
* Data analysis

## How to Run

```bash
python main.py
```
